# 9、排列硬币

```js
// 总共有 n 枚硬币，将他们摆成一个阶梯形状，第 k 行就必须正好有 K 枚硬币
// 给定一个数字 n, 找出可形成完整阶梯行的总行数
// n 是一个非负整数，并且在 32 位有符号整型的范围内
```

```js
// 方式一： 迭代
function arrangeCoins(n) {
  for(let i = 0; i<=n; i++) {
    n = n-i;
    if(n <= i) {
      return i
    }
  }
}

console.log('arrangeCoins--', arrangeCoins(8));
```
